graoJS |version| Manual
=======================

Welcome to graoJS Manual. graoJS_ is NodeJS_ framework for
build database driven applications.

graoJS_ was bring to life by Synack_ (see also `Synack GitHub Acccount`_).

In this manual you will find a full range of features and references for starting
and mastering on graoJS.

This is a working-in-progress. Feel free to help us on writing, fixing or translating:
fork the docs repository and contact us for pull request.

Manual repository: `graoJS docs`_

Doc Maintainer: Rafael Goulart (`rafaelgou GitHub Account`_).
